# 🚀 Active Directory Automation Dashboard

## 📖 Overview

This project is a PowerShell-based Active Directory Management Dashboard designed to simplify Windows Server administration and automate common domain management tasks.

The solution provides a centralized web-based control panel that allows administrators to monitor domain computers, track user activity, and perform remote administrative actions from a single interface.

What started as a simple PowerShell automation project evolved into a lightweight NOC-style management platform that improves operational efficiency and reduces repetitive administrative work in enterprise environments.

---

## 🎯 Project Objectives

* Automate repetitive Active Directory administration tasks.
* Centralize domain management into a single dashboard.
* Improve visibility across domain-joined devices.
* Simplify remote workstation administration.
* Provide real-time monitoring of computer and user activity.
* Demonstrate the power of PowerShell beyond traditional scripting.
